## Schemes

Both EVM schemes defined by x402 are registered on every supported 0G network:

- **exact** — the buyer authorizes and pays the advertised price.
- **upto** — the buyer authorizes a maximum; the seller settles for the actual usage.

Use [`GET /supported`](https://facilitator.heraldprotocol.xyz/supported) to query the authoritative `(network, scheme)` list at runtime.

## Pricing in USD

When you use `ExactEvmScheme` from `@heraldprotocol/x402/server`, you can price endpoints in USD strings like `"$0.001"`. The scheme resolves the amount against the default stablecoin for the declared network — no hardcoded decimals or token addresses required.

```ts
import { ExactEvmScheme } from "@heraldprotocol/x402/server";

new x402ResourceServer(facilitator).register(
  "eip155:16661",
  new ExactEvmScheme()
);
```
